## 🧠 SISTEMA DE APRENDIZAJE

### Método LRE (Leer-Resaltar-Explicar)

El método que garantiza comprensión, no memorización.

#### PASO 1: LEER (25 minutos)

```markdown
├─ Lee documentación EN INGLÉS (no traduzcas párrafos)
├─ Fuentes: es.javascript.info | The Odin | FullStackOpen | MDN
├─ No tomes notas largas (solo palabras clave)
└─ Objetivo: Entender el CONCEPTO, no memorizar sintaxis
```

#### PASO 2: RESALTAR (15 minutos)

```markdown
├─ Identifica 5-10 palabras técnicas clave de la lectura
├─ Ejemplo:
│  └─ closure = envolvimiento de variables
│  └─ promise = promesa de resultado futuro
│  └─ async = operación asíncrona
│  └─ callback = función de retorno
│  └─ prototype = plantilla de objeto
└─ Escribe estos términos en tu cuaderno digital
```

#### PASO 3: EXPLICAR (20 minutos)

```markdown
├─ Abre ChatGPT/Gemini
├─ Prompt: "Explica [concepto] en inglés simple para principiante"
├─ Lee la explicación EN INGLÉS
├─ Traduce TU COMPRENSIÓN al español (no traduzcas la IA)
├─ ESCRIBE el concepto en código JavaScript
└─ Guarda en tu carpeta de la semana
```

### Regla de los 15 Minutos (Para cuando te bloquees)

```markdown
⏱️  0-5 minutos:
    └─ Lee el error exacto
    └─ Intenta 1-2 soluciones

⏱️  5-10 minutos:
    └─ Busca el error en Google EN INGLÉS
    └─ Revisa StackOverflow

⏱️  10-15 minutos:
    └─ Pregunta a ChatGPT con contexto exacto del código

⏱️  15+ minutos:
    └─ VE LA SOLUCIÓN DE REFERENCIA
    └─ ENTIENDE cada línea
    └─ ESCRIBE de nuevo SIN COPIAR

JAMÁS: Copiar sin entender
SIEMPRE: Entender antes de implementar
```

### Cardio-Learning (Aprendizaje paralelo)

```markdown
Mientras haces cardio (caminar, correr, gym):

Semana 1-4:   Escucha "Syntax.fm" podcast (JavaScript semanal)
Semana 5-10:  Escucha "Frontend Masters" recordings
Semana 11-14: Escucha "The Changelog" (industry news)
Semana 15+:   Escucha "Shoptalk Show" (web development)

BENEFICIO: Tu subconsciente procesa conceptos 
          mientras entrenas físicamente
```

### JUEVES: Semana 1 — Día 4

**Tema Principal:** Flexbox Fundamentals

#### INMERSIÓN (0:50)
- **Recurso:** The Odin Project → Foundations → Flexbox
- **Lectura:**
  - "Introduction to Flexbox" (15 min)
  - "Growing and Shrinking" (20 min)

**Palabras Clave:**
- Flexbox = contenedor flexible
- Main Axis = eje principal (horizontal por defecto)
- Cross Axis = eje cruzado
- justify-content = alineación en main axis
- align-items = alineación en cross axis

#### CONSTRUCCIÓN (1:10)

```html
<!-- fase-1/semana-01/flexbox-practice.html -->
<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Flexbox Practice</title>
    <style>
        * {
            margin: 0;
            padding: 0;
            box-sizing: border-box;
        }

        body {
            font-family: Arial, sans-serif;
            padding: 2rem;
            background-color: #f5f5f5;
        }

        h2 {
            margin-top: 2rem;
            margin-bottom: 1rem;
            color: #2c3e50;
        }

        /* FLEXBOX HORIZONTAL */
        .flex-horizontal {
            display: flex;
            justify-content: space-around;
            align-items: center;
            gap: 2rem;
            background-color: #ecf0f1;
            padding: 2rem;
            margin-bottom: 2rem;
            border-radius: 8px;
            min-height: 150px;
        }

        /* FLEXBOX VERTICAL */
        .flex-vertical {
            display: flex;
            flex-direction: column;
            gap: 1rem;
            background-color: #ecf0f1;
            padding: 2rem;
            margin-bottom: 2rem;
            border-radius: 8px;
        }

        /* FLEXBOX CON WRAP */
        .flex-wrap {
            display: flex;
            flex-wrap: wrap;
            gap: 1rem;
            background-color: #ecf0f1;
            padding: 2rem;
            border-radius: 8px;
        }

        /* ITEMS */
        .flex-item {
            background-color: #3498db;
            color: white;
            padding: 2rem;
            flex: 1;
            text-align: center;
            border-radius: 8px;
            min-width: 100px;
        }

        .flex-wrap .flex-item {
            flex: 0 1 calc(33.333% - 1rem);
        }

        /* RESPONSIVE */
        @media (max-width: 768px) {
            .flex-horizontal {
                flex-direction: column;
            }

            .flex-wrap .flex-item {
                flex: 0 1 calc(50% - 1rem);
            }
        }
    </style>
</head>
<body>
    <h1>Flexbox Mastery</h1>

    <h2>1. Distribución Horizontal (space-around)</h2>
    <div class="flex-horizontal">
        <div class="flex-item">Item 1</div>
        <div class="flex-item">Item 2</div>
        <div class="flex-item">Item 3</div>
    </div>

    <h2>2. Distribución Vertical (flex-direction: column)</h2>
    <div class="flex-vertical">
```
